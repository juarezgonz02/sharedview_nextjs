# Sharedview Next JS Version

Remake for original Sharedview proyect made back in 2020 

### Requires
- Node ^18.2
- Windows/Linux 
- FFmpg (https://www.ffmpeg.org/)

### Contains
- RMTP Server 
- NextJS WebSite 
- WebSocket Server for signaling between peers
- (Comming soon) API Server
